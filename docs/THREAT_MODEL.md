## החלטות מדיניות (מאושרות מהתוכנית)

- **Default deny**: מאושר → פתוח. לא מאושר → חסום. ספק → חסום.
- שרת לא זמין → policy cache מקומי.
- אין policy מקומי → חסימת גלישה, ערוץ תמיכה מינימלי נשאר פתוח.
- פרטיות: שומרים רק מטא-דאטה (דומיין שנחסם, אפליקציה, סיבה, סטטוס). **לעולם לא** תוכן הודעות / תמונות / סיסמאות / תוכן בנקאי.

## מה המערכת מגינה מפניו (לפי סדר עדיפות)

1. גלישה לאתרים לא מאושרים (דפדפן + WebView) — דרך DNS/SNI filtering
2. שימוש באפליקציות לא מאושרות — דרך App Monitor + block overlay
3. התקנת אפליקציות עוקפות (APK, clone, dual app) — דרך app verification (שלב 4)
4. כיבוי/שיבוש ההגנה (כיבוי VPN, ביטול הרשאות, הסרה) — דרך detection + fail closed (שלב 5), ובסוף Device Owner (שלב 6)
